# Introducción -> MySQL A Terminal (por pasos)

---

<br>

Vamos a la terminal. Ahí aplicamos lo siguiente:

- Abro la Terminal
- Navegamos hasta el archivo que queramos abrir con MySQL.

<br>


- Introducimos el comando <code>mysql</code> <code> -u </code>  <code> *[nombreArchivo.sql]* </code> <code> -p </code>

- <code> -p </code> : me pedirá la contraseña de acceso a la base de datos.

--> Introduzco la contraseña **¡ya estoy dentro!**
